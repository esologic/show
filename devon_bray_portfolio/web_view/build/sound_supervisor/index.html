Sound Supervisor (September of 2016)

Adds physical controls to the Windows volume mixer.

Completed as a hackathon (hackMIT) project, this device maps the individual application volumes for programs running on Windows to potentiometer knobs. User can quickly adjust these volumes without having to open the volume mixer window.
